## 工程

### 1. 床にジョイントマットを引く

まずは床を傷つけないようにジョイントマットを引きます。

cf. https://www.amazon.co.jp/dp/B01N9TS66N

{{< figure src="yuka-before.jpg" class="left" width="400" >}}

{{< figure src="yuka-after.jpg" class="left" width="400" >}}

### 2. 木材を用意する

次は木材を用意します。

今回は外が滑らかな MDF を選択し、サイズは成人男性がすっぽり入れるぐらいにしました。

- MDF ボード(壁) 1825mm×900mm×12mm × 4
- MDF ボード(天井) 900mm×900mm×12mm
- MDF ボード(机) 450mm\*900mm\*12mm

材木屋ざいいちさんで注文しました。
好きなサイズでカットできるし、お家の前まで届けてくれてありがたい！

cf. https://www.zaiichi.com/

### 木材を L 字金具と蝶番で繋げる

まず、壁となる木材を L 字金具で繋ぎ止めます。

これが本当に大変だった。皆様には手伝ってくれる友達を作ることをおすすめします。

cf. [L 字金具](https://www.amazon.co.jp/dp/B0FB8DXYXV)

cf. [電動ドライバー](https://www.amazon.co.jp/dp/B0DLFQJ6FW)

{{< figure src="driver.jpg" class="left" width="300" >}}
{{< figure src="eru.jpg" class="left" width="200" >}}

壁となる木材のうち一枚はドアとして開閉させる必要があるため、蝶番を用います。

これに屋根を載せて、ベースとなる部分は完成です！

cf. [蝶番](https://www.amazon.co.jp/dp/B0D3HZWLZ6)

cf. [ドアハンドル](https://www.amazon.co.jp/dp/B0DSCC859C)

### 3. コーキングガンで木材の隙間を埋める

ネジで木材を繋ぎ留めたら、その隙間をコーキングで埋めます。

ゴムパッキンのような質感です。

cf. [コーキングガン](https://www.amazon.co.jp/dp/B00N3TWIVE)

cf. [コーキング剤](https://www.amazon.co.jp/dp/B002YMQGNM)

{{< figure src="caulking.jpg" class="left" width="300" >}}

### 4. 遮音材を壁に貼る

音を反射する「遮音」と、音を減衰させる「吸音」でそれぞれ役割が異なるようです。

音源 → 吸音材 → 遮音材 → 壁 のような順番にすると、声がまず吸音材で減衰され、吸収しきれなかった音も遮音材や壁によって反射され、別の吸音材に吸収されて効果的であるようです。

よってまず壁に遮音材をぺたぺた貼ります。

接着にはグルーガンという熱で接着剤を溶かす道具を用います。

cf. [遮音シート](https://www.amazon.co.jp/dp/B09VGJTRWZ)

cf. [グルーガン](https://www.amazon.co.jp/dp/B0F7HQCDC3)

{{< figure src="syaon.jpg" class="left" width="300" >}}

{{< figure src="syaon-peta.jpg" class="left" width="300" >}}

### 5. 吸音材を遮音材の上から貼る

次に音を減衰させる吸音材を貼ります。

歌ってみた収録では音の反響を抑えることにも活躍するそうです。

最後に机とかつけて完成！！

cf. [吸音材](https://www.amazon.co.jp/dp/B09GFC7HLS)
